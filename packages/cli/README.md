## Getting Started ðŸš€

Install Theta CLI.

```sh
$ dart pub global activate theta_cli
```

Link a project by its key.

```sh
$ theta link -k <anon key>
```

You can generate files from design by running:

```sh
$ theta gen
```

### Other commands

```sh
# Unlink a project
$ theta unlink

# Show CLI version
$ theta --version

# Update CLI
$ theta update

# Show usage help
$ theta --help
```
