# Ottenere il token API del progetto

<YouTube id="vIE4gZZ8ufo"/>

Per recuperare il token API dal tuo progetto, segui i passaggi riportati di seguito:

1. **Crea un nuovo progetto:** Vai su [app.buildwiththeta.com](https://app.buildwiththeta.com) e crea un nuovo progetto.

2. **Apri l'Editor del Progetto:** Dopo aver creato il progetto, accedi all'editor del progetto.

3. **Fai clic sul nome del progetto:** Troverai il nome del progetto nell'angolo in alto a sinistra, accanto al logo di Theta. Fai clic su di esso.

4. **Visualizza la chiave API:** Dopo aver fatto clic sul nome del progetto, apparirà una scheda inferiore che mostra il campo 'API key'. Questo è il tuo token API.

Questa chiave anonima (anon) è ciò che utilizzerai per connettere la tua applicazione client a Theta, utilizzando la nostra libreria client Flutter. Assicurati di archiviare questa chiave anonima in modo sicuro, preferibilmente in un file di ambiente (`.env`), per evitare accessi non autorizzati.

Tieni presente che le applicazioni client possono connettersi al tuo progetto solo se è impostato su 'pubblico'. Assicurati di avere configurato correttamente la visibilità del tuo progetto.
