# Come eseguire l'app di esempio di Theta in Flutter

Per avviare l'app di esempio di Theta in Flutter basta andare nella cartella example all'interno del pacchetto builder. 
Potete quindi eseguire il seguente comando nel vostro terminale:

```bash
flutter run -d chrome
```

## Configurazione del progetto

1. Nella riga 9, cambia il placeholder 'thetaKey' con la tua chiave anonima.
2. Nella riga 35, cambia il nome del UIBox con il nome del componente che desideri visualizzare. Per impostazione predefinita, tutti i primi componenti di ogni progetto sono denominati homepage.

---

Eseguendolo, vedrai una pagina vuota con il componente creato all'interno di Theta. Con questa demo puoi vedere come si comporta all'interno di un'app Flutter, provare le modalità di risposta e cercare di mixare il tuo design con il codice personalizzato attraverso i workflow (troverai gli esempi già scritti all'interno di main.dart)

<Warning>Attenzione: in questo progetto di test, la cache è stata disabilitata nella riga 8. Tuttavia, è consigliabile tenerla sempre attiva nei tuoi progetti per evitare di ricostruire l'applicazione troppe volte.</Warning>

Buon divertimento!
