# Proje API Anahtarını Al

<YouTube id="vIE4gZZ8ufo"/>

Projeden API anahtarını almak için aşağıdaki adımları izleyin:


1. **Yeni Bir Proje Oluşturun:** [app.buildwiththeta.com](https://app.buildwiththeta.com) adresine gidin ve yeni bir proje oluşturun.

2. **Proje Editörünü Açın:** Proje oluşturulduktan sonra, proje editörüne giriş yapın.

3. **Proje Adına Tıklayın:** Proje adını Theta logosunun yanında, sol üst köşede bulacaksınız. Üzerine tıklayın.

4. **API Anahtarını Görüntüleyin:**  Proje adına tıkladığınızda, bir alt sayfa açılacak ve 'API key' alanını içeren bir bölüm görüntülenecektir. Bu, API anahtarınızdır.

Bu anonim (anon) anahtarı, Theta'ya istemci uygulamanızı bağlamak için kullanacaksınız. Flutter istemci kütüphanemizi kullanarak. Bu anonim anahtarını güvenli bir şekilde sakladığınızdan emin olun, tercihen bir ortam (`.env`) dosyasında, yetkisiz erişimi engellemek için.

Unutmayın, istemci uygulamaları yalnızca projeniz 'public' olarak ayarlandığında projenize bağlanabilir. Proje görünürlüğünü uygun şekilde yapılandırdığınızdan emin olun.
