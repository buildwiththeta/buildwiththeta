# Compatibility

Platforms supported by the [Theta Flutter library](https://pub.dev/packages/theta).

<Warning>Theta is only compatible with Dart 3.</Warning>

## Supported platforms

- [x] Web
- [x] iOS
- [x] Android
- [x] macOS
- [x] Windows

---

## Configuration

### Web

No configuration is required.

### iOS

No configuration is required.

### Android

> Time required for setup: 2 minutes.

<Warning>Theta is not be available to users running Android SDKs below 19.</Warning>

Since Theta depends on `[package_info_plus](https://pub.dev/packages/package_info_plus)` the `minSdkVersion` must be at least 19.

```
android/app/build.gradle:
android {
  defaultConfig {
    minSdkVersion 19
  }
}
```

Also, Theta requires [internet connection](https://developer.android.com/training/basics/network-ops/connecting).

```
<manifest xmlns:android...>
  ...
  <uses-permission android:name="android.permission.INTERNET" />
  <application ...
</manifest>
```

### macOS

> Time required for setup: 1 minute.

macOS apps must allow network access in the relevant .entitlements files.

```
<key>com.apple.security.network.client</key>
<true/>
```

Learn more about [setting up entitlements](https://docs.flutter.dev/platform-integration/macos/building#setting-up-entitlements).

### Windows

No configuration is required.
