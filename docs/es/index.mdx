# Build with Theta
[Theta](https://buildwiththeta.com) es una herramienta open source que simplifica la creación y prueba de aplicaciones con interfaces de usuario dinámicas, sin la necesidad de reconstruirlas.

## Para empezar
Aprende cómo crear un proyecto de interfaz de usuario en Theta y a transmitir un componente dentro de tu código.

### Lo que aprenderás
- Cómo crear un nuevo proyecto en Theta.
- Cómo instalar el paquete Theta dentro de tu aplicación.
- Cómo transmitir un componente de la interfaz de usuario dentro de tu código.
