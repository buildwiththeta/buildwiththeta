# Migración de Teta a Theta
Si actualmente estás utilizando Teta y quieres migrar tus datos a Theta, ofrecemos un servicio dedicado de migración de datos para hacer que la transición sea lo más fácil posible. Ten en cuenta que aunque Theta y Teta comparten algunas similitudes, Theta fue creado para un propósito diferente y tiene un alcance distinto.

## Para empezar

Para solicitar la migración de datos desde Teta a Theta, por favor visita https://buildwiththeta.com/migration y rellena el formulario con la siguiente información:

- Nombre y apellidos
- Email
- IDs de los proyectos (separados por comas)

Para encontrar los IDs de tus proyectos, simplemente abre un proyecto en Teta y copia el ID numérico desde la URL, que debería verse algo como esto: .../editor/$projectId/...

❗ Por favor ten en cuenta que la migración de datos podría no ser posible para todos los proyectos, ya que Theta utiliza una estructura de base de datos diferente y algunos nodos podrían no ser utilizados en el nuevo producto. Revisaremos cada solicitud de migración caso por caso para determinar si es posible.

## Plazo de migración
Por favor, ten en cuenta que el proceso de migración de Teta a Theta puede llevar algún tiempo para completarse. Debido a las diferencias en las estructuras y funcionalidades entre las dos plataformas, el proceso de migración requiere una planificación y ejecución cuidadosas. Nos esforzamos por hacer la transición lo más fluida posible, pero por favor comprende que puede que no sea un proceso inmediato.

## Lo que se puede migrar

Los siguientes datos pueden ser migrados de Teta a Theta:

- Proyectos
- Canales para cada proyecto
- Páginas y componentes para cada canal
- Nodos para cada página o componente

## Motivos de rechazo

Por favor ten en cuenta que nos reservamos el derecho de rechazar solicitudes de migración por los siguientes motivos:

- El nombre y correo electrónico proporcionados no coinciden con los datos guardados en Teta
- IDs de los proyectos incorrectos 
- Proyecto considerado inadecuado o inapropiado para la migración
- Contenido inapropiado dentro de los proyectos
- Imposibilidad técnica de la migración por razones específicas

Gracias por considerar a Theta como su nueva plataforma, y esperamos poder ayudarte con el proceso de migración de datos.
