# Teams
**Theta utiliza teams para proporcionar experiencias personales y colaborativas.
Cada equipo puede estar compuesto por un solo miembro o varios usuarios.**

Un team personal se puede utilizar para proyectos de desarrollo individuales y un team multiusuario para la colaboración con otros miembros.
Cada proyecto pertenece a un equipo, y cada proyecto puede tener diferentes ramas para diferentes entornos o versiones.

Para compartir el team personal con otros miembros, simplemente invítalos por correo electrónico desde el menú "Team Settings".
Todos los proyectos se compartirán automáticamente con los miembros invitados.
