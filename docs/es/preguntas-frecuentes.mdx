## Sobre Theta

### Qué es Theta.

Theta, una herramienta alojada de código abierto, facilita el proceso de desarrollo, gestion y pruebas de aplicaciones al permitir cambios en la UI remota sin necesidad de reconstruir la aplicación o realizar configuraciones complejas.

### ¿Por qué usar Theta?

Diseñado para facilitar el diseño rápido de interfaces de usuario, Theta simplifica la creación de interfaces dinámicas directamente dentro de las aplicaciones. Theta no es un constructor de aplicaciones sin código. En su lugar, permite a los desarrolladores incorporar Theta en sus flujos de trabajo existentes, asegurando una integración perfecta sin comprometer la calidad del código. Con Theta, puedes diseñar una barra de aplicaciones receptiva y mostrarla en una aplicación Flutter en cuestión de minutos.

### ¿Cómo usar Theta?

Aprende en [este video](https://youtu.be/oFed0NIqBZI) cómo crear un diseño de UI en Theta y incorporar un componente sin problemas en tu código.

### ¿Puedo registrarme en Theta?

## Preguntas técnicas

### ¿Es adecuado para stores móviles?

Claro, nuestro sistema es adaptable a cualquier tipo de stores, sin embargo, se debe utilizar con moderación y sin abusar de él. También puedes utilizar Theta en cualquier plataforma compatible con Flutter.

### ¿Qué plataformas admite Theta?

Aunque Theta es una aplicación web que se puede utilizar desde un navegador, ofrecemos una integración oficial para Flutter. Esto te permitirá utilizar interfaces remotas creadas en app.buildwiththeta.com directamente dentro de tus aplicaciones Flutter. En cuanto a la compatibilidad, el paquete de Flutter es compatible con todas las plataformas admitidas por Flutter (iOS, Android, Web, Desktop).

### ¿Es seguro utilizar Theta en mi aplicación?

Theta sólo se preocupa de transmitir UIs remotas y estáticas dentro de tu app, con una configuración mínima requerida. Esto le permite evitar el push de código, que podría comprometer seriamente las aplicaciones que ya están en producción.
En lo que respecta a los stores, es seguro usar Theta con moderación, sin abusar de los cambios completos de UI.

### ¿Es posible utilizar Theta para crear una aplicación?

Theta no es un constructor de aplicaciones, por lo que no está diseñado para crear una aplicación completa desde cero. En cambio, está diseñado para ser utilizado en el proceso de creación de una aplicación, para facilitar la creación de interfaces de usuario responsivas y simplificar en gran medida el proceso de actualización de la aplicación.

### ¿Puedo usar más de un `UIBox` por página??

Por supuesto. El `UIBox` se refiere a un único componente, sin ningún background, que puede ser insertado dentro de una página junto con otros UIBoxes para crear interfaces complejas.

### ¿Puedo decidir el color de fondo de un componente dentro de Theta?

No, los componentes no tienen un fondo predeterminado. Sin embargo, puedes insertar uno añadiendo un contenedor de fondo del tamaño de la pantalla.
