## Events section

![https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/conversion_events_panel.png](https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/conversion_events_panel.png)

The events section is for keeping track of logs of events triggered by users in your apps.

### Conversion events

Conversion events are a simple yet powerful way to track user actions in your app. With Build with Theta, we provide a straightforward conversion events system that allows you to easily monitor and analyze user interactions, all while prioritizing simplicity and ease of use.

[Read more about conversion events.](https://docs.buildwiththeta.com/en/studio/conversion_events)
